## Assignment Overview

This project is developed as part of the **Gushwork Web Developer Assignment**.

The goal was to build a **pixel-perfect, responsive web page** using **vanilla HTML, CSS, and JavaScript**, strictly following the provided Figma design.

---

## Features Implemented

### Sticky Header

- Appears after scrolling past the first fold
- Smooth transition animation
- Hides when scrolling back up
- Positioned above navigation as per Figma

### Image Carousel with Zoom

- Interactive image slider
- Hover-based zoom preview functionality
- Smooth transitions and animations
- Thumbnail navigation support

### Fully Responsive Design

- Desktop, Tablet, Mobile optimized
- Flexbox & Grid used for layout
- Matches Figma responsiveness

---

## Tech Stack

- HTML5 (Semantic structure)
- CSS3 (Flexbox, Grid, responsive design)
- Vanilla JavaScript (DOM manipulation, interactions)
